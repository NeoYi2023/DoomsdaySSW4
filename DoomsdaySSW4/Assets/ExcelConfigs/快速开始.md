# Excel配置表转JSON工具 - 快速开始

## 步骤1：准备Excel文件

1. 在Excel中创建配置表，按照 `README.md` 中的格式要求填写数据
2. 将Excel文件另存为CSV格式：
   - 文件 > 另存为
   - 选择"CSV UTF-8（逗号分隔）(*.csv)"
   - 保存到 `Assets/ExcelConfigs/` 目录
   - 文件名格式：`配置表名_中文备注字段.csv`

## 步骤2：使用转换工具

1. 在Unity Editor菜单栏选择：`Tools -> 配置表 -> Excel转JSON`
2. 工具窗口会打开，显示转换进度和日志
3. 点击"转换所有Excel文件"按钮
4. 等待转换完成

## 步骤3：验证结果

转换后的JSON文件会保存到：
```
Assets/Resources/Configs/
```

文件名：
- `TaskConfigs.json`
- `OreConfigs.json`
- `OreSpawnConfigs.json`
- `DrillConfigs.json`
- `ShipConfigs.json`
- `EnergyUpgradeConfigs.json`

## 常见问题

### Q: 为什么使用CSV而不是直接读取Excel？
A: Unity Editor环境对Excel库的支持有限，使用CSV格式更简单可靠。Excel可以轻松导出为CSV格式。

### Q: CSV文件中文乱码怎么办？
A: 确保Excel导出时选择"CSV UTF-8"格式，而不是普通的CSV格式。

### Q: 转换失败怎么办？
A: 检查：
- 文件名格式是否正确（配置表名_中文备注字段.csv）
- 表头字段名是否与C#类属性名完全匹配
- 数据类型是否正确（整数、布尔值、枚举值等）
