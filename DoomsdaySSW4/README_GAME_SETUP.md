# 游戏运行设置指南

## 概述

本文档说明如何检查游戏运行所需的配置，以及如何修复缺失项。

## 快速开始

### 1. 使用自动化检查工具（推荐）

1. 打开Unity编辑器
2. 打开GameScene场景
3. 在菜单栏选择：`Tools > 游戏运行检查`
4. 点击 `开始检查` 按钮
5. 查看检查报告并修复所有 ❌ 红色问题

### 2. 查看详细检查清单

打开 `GAME_RUNTIME_CHECKLIST.md`，按照清单逐项检查。

### 3. 修复问题

打开 `FIX_GUIDE.md`，根据问题类型查找对应的修复步骤。

## 工具说明

### 游戏运行检查工具
- **位置**：`Tools > 游戏运行检查`
- **功能**：自动检查场景配置、UI结构、组件引用、配置文件等
- **输出**：详细的检查报告，包含通过项、警告和问题

### 中文字体设置工具
- **位置**：`Tools > 设置中文字体`
- **功能**：批量设置TextMeshPro中文字体
- **使用**：选择字体资源，点击按钮批量应用

### 字体诊断工具
- **组件**：`FontDiagnostics`
- **功能**：诊断字体配置和字符支持
- **使用**：添加到场景中的GameObject，运行游戏自动诊断

## 检查流程

### 第一步：运行自动化检查
```
Tools > 游戏运行检查 > 开始检查
```

### 第二步：修复高优先级问题
按照检查报告，优先修复所有 ❌ 红色问题：
1. GameInitializer GameObject和组件
2. Canvas和EventSystem
3. GameScreen基本UI结构
4. UI组件引用连接

### 第三步：验证修复
1. 点击Play运行游戏
2. 检查Console日志
3. 测试核心功能

### 第四步：完善功能
修复 🟡 中优先级问题，确保功能完整。

## 常见问题

### Q: 检查工具显示很多问题，从哪里开始？
**A:** 按照优先级修复：
1. 首先修复 ❌ 红色问题（阻止运行）
2. 然后修复 ⚠ 黄色警告（影响功能）
3. 最后优化 🟢 绿色项（可选）

### Q: 如何知道某个配置是否正确？
**A:** 
- 使用自动化检查工具
- 或参考 `GAME_RUNTIME_CHECKLIST.md` 中的详细说明

### Q: 修复后如何验证？
**A:**
1. 重新运行检查工具
2. 运行游戏测试功能
3. 查看Console日志

### Q: UI结构太复杂，有没有快速创建方法？
**A:**
- 参考 `GAME_RUNTIME_CHECKLIST.md` 中的详细步骤
- 或使用 `DEMO_SETUP_GUIDE.md` 中的指南
